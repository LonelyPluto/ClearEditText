# Sample
用法和普通的EditText相同，把自定义的路径写对就行
``` xml
<com.lonelypluto.clearedittext.ClearEditText
        android:layout_width="match_parent"
        android:layout_height="wrap_content"
        android:textSize="15sp"
        />
```
